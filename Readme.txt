Embed a customized google map with a customized marker. Fast and easy.

== Description ==
The Custom Google Maps plugin is the easiest way for adding a Google Map with your own customized map marker to your Website. Just by using a shortcode [cgm lat="" long=""].
The plugin has a settings page and allows you to easily add a Google Maps API. The shortcode has optional attributes: width, height, scrollwheel, zoom and controls. This plugin is a beta version, and will be actively developed.
** WARNING:** This plugin is a beta version


== Installation ==
1. Unpack and Upload the custom-google-maps.zip plugin in the to the "/wp-content/plugins/" directory, or upload the plugin zip via the plugins settings page.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Add your Google Maps API code on the settings page
4. Add the shortcode [cgm lat="" long=""] where you want on your WordPress website.
